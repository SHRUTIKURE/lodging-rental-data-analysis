# lodging-rental-data-analysis

## Project Description

This project analyzes lodging rental data using Python. The dataset contains information about different lodging properties such as their name, category, type, rating, price, and average revenue.
The goal of this project is to load the dataset, clean and organize the data, analyze trends, and visualize insights using Python libraries.

## Dataset

The data used in this project is stored in a .dat file.
A .dat file is a data file that stores serialized Python objects. In this project, the file is loaded using the pickle library.
The dataset includes information such as:
1. Date
2. Property Name
3. Category
4. Type of lodging
5. Rating
6. Price
7. Average revenue

## Libraries Used

This project uses the following Python libraries:
1. pandas – data manipulation and analysis
2. numpy – numerical operations
3. matplotlib – data visualization
4. seaborn – statistical data visualization

## Key Features

1. Loading dataset from a .dat file
2. Data cleaning and preprocessing 
3. Object-Oriented Programming using a Lodging class
4. Data analysis using pandas
5. Data visualization using matplotlib and seaborn
6. pickle – loading the .dat dataset
7. gdown – downloading the dataset
